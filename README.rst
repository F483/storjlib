storjterms reference implementation.


Setup
#####

Install

::

    $ sudo apt-get install python-dev gcc
    $ sudo pip install storjnode
    $ storjnode --help


Update

::

    $ sudo pip install storjnode --upgrade
    $ storjnode --help
